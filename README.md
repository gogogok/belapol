<h2>Телеграм</h2>
Для парсинга телеграмма используются Python скрипты, а json с готовыми данными отдаёт локальный сервер. Есть фолбэк на json в проекте, поэтому работать будет.

Чтобы телеграм парсинг нормально работал, нудно запустить сервер (python3 -m http.server 8000), потом скрипты (python3 load_channel.py,  python3 border_posts_load.py
)

Также рабоать будет только на симуляторе, так как сервер локальный. Чтобы работало везде, нужно просто сделать нормальный сервер, но пока извините, но нет.

Так как скрипты содержат конфеденциальную информацию, я не могу их выложить в открытый доступ. Поэтому для получения скриптов можете написать в мой тг *@noncrit*

<h2>Использование VPN</h2>

Загрузка билетов на автобусы и новостей с сайтов не будет работать, так как симулятор плохо переносит VPN.
